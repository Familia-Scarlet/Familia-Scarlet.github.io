    C++

    const

    const一般用在代码前面表示常量，只可读不可改，这里用在函数中表示常成员函数： 只读函数，只可读不可改数据成员的值。也就是时候const定义的函数里面的数据成员 的值不可改变

    重载单目运算符++

    前置++的效率好，因为没有临时对象的产生

    A& operator++()//++i 前置++实现
{
    ++x;	//先自增
    ++y;
    return *this;	//后引用
}
A operator++(int)	//i++ 后置++实现
{
    //int参数没有任何意义，只是为了区分是前置还是后置形式
    A a = *this;	//保存对象引用
    ++(*this);	//自增，调用前面实现的前置++
    return a;	//返回先前保存的对象
}

    WebAPI

    WebAPI学习

    基本认知

    1. DOM(文档对象模型)：包含各种HTML标签对应的JS对象，用于操作网页内容，可实现网页内容特效和用户交互。

    2. DOM树：将HTML文档以树形结构表现出来的一棵树，体现了标签与标签的关系。

    3. DOM对象：浏览器根据HTML文档生成的JS对象，有其相应的属性，与标签一一对应。

    4. document对象：DOM中提供的一个对象，是最大的对象，网页全部内容都在里面。该对象提供了操作网页内容的方法。

    5. BOM(浏览器对象模型)：整个window、navigator、location、document、history、screen对象组成浏览器对象模型，其中window是最大的BOM对象。

    6. window对象：浏览器内置的全局对象，BOM中最大的对象，所有的WebAPIs均基于window对象，window对象下包含了navigator、location、document、history、screen对象。

      • navigator：记录浏览器设备的信息。

      • location：控制浏览器页面位置。

      • document：记录浏览器内容结构。

      • history：记录浏览器浏览记录。

      • screen：记录用户的屏幕信息。

    数据库

    1NF、2NF、3NF、BCNF

    部分函数依赖

    X–>Y，X'是X的真子集，若满足X'–>Y，那么Y部分函数依赖于X，这就是部分函数依赖。

    完全函数依赖

    X–>Y，X'是X的真子集，若满足X'!–>Y，那么Y完全函数依赖于X，这就是完全函数依赖。

    1NF

    字段不可分，且不存在重复字段

    2NF

    有主键，且非主键字段完全函数依赖于主键，不能部分函数依赖于主键

    3NF

    在2NF基础上，消除非主键函数依赖于非主键

    计算机网络

    Https工作流程

    1. 浏览器向服务器发送对称加密套件、非对称加密套件、浏览器随机数
    2. 服务器向浏览器发送对称加密套件、非对称加密套件、服务器随机数、CA证书
    3. 浏览器收到CA证书后，对CA证书进行校验，使用浏览器内置的CA公钥对CA证书的数字签名进行解密，将解密结果和CA证书中的明文内容进行比较，如果相同则可以认为这个CA证书是可信的。然后取出CA证书中的服务器公钥，用公钥加密pre-master随机数，然后发送给服务器。
    4. 这个时候浏览器和服务器都有了三个随机数，分别是浏览器随机数、服务器随机数、pre-master随机数，用这三个随机数生成对称密钥，这个对称密钥将在后续进行数据加密和解密使用。
