    Hadoop学习笔记

    大数据概念

    海量、高增长率和多样化的信息资产

    数据储存单位是TB、PB、EB以上

    大数据主要解决海量数据的采集、存储和分析计算问题

    大数据特点

    4个V

    1. Volume

      海量数据

    2. Velocity

      数据产生快，比如天猫双十一96s内，交易额达到100亿

    3. Variety

      数据类型多，有结构化数据和非结构化数据，以数据库、JSON文档代表的结构化数据，以网络日志、音视频、图片、地理信息为代表的非结构化数据

    4. Value

      价值密度的高低与数据总量的大小成反比，数据量庞大但价值可能不高。所以，需要进行数据清洗，将无效数据、低价值数据过滤掉。

    什么是微服务？

    微服务是一种经过良好架构设计的分布式架构方案，微服务架构特征：

    lambda表达式

    Lambda 表达式的使用前提:

    • 必须有接口（不能是抽象类），接口中有且仅有一个需要被重写的抽象方法。
    • 必须支持上下文推导，要能够推导出来 Lambda 表达式表示的是哪个接口中的内容，也就是说将lambda表达式赋值或传递给接口类型的参数时，需要能唯一确认重写的接口的抽象方法。
    public class LambdaTest {
    final static String hello = "Hello, ";
    private static String goodbye = "Goodbye, ";
    
    public static void main(String[] args) {
    
    	//被实现的抽象接口
    	interface GreetingMessage {
        	void sayMessage(String message);
    	}
    
    
    	//第一个lambda
        GreetingMessage message1 = (String message) ->
        System.out.println(hello + message);
        
        //第一个的匿名内部类全写
        GreetingMessage message1 = new GreetingMessage(){
        	@Override
        	void sayMessage(String message){
        		System.out.println(hello + message);
        	}
        }
        
        
        //第二个lambda
        GreetingMessage message2 = (message) ->
        {
            //goodbye = "the time is late, I'd go now, goodbye";
            goodbye += "the time is late, I'd go now, ";
            System.out.println(goodbye + message);
        };
        
        message1.sayMessage("张三");
        message2.sayMessage("李四");
    }
    

}

    html笔记

    语义元素（semantic elements）

    顾名思义，就是能表达出意义的元素，可以用来指明导航、头部、主体、侧边栏以及尾部。这些语义元素也都是块元素。

    • <header>元素描述了文档的头部区域

    • <nav>标签定义导航链接的部分。

    • <section>标签定义文档中的节（section、区段）。比如章节、页眉、页脚或文档中的其他部分。

    • <article>标签定义独立的内容。

    • <aside>标签定义页面主区域内容之外的内容（比如侧边栏）。

    • <figcaption>标签定义 <figure> 元素的标题，该元素应该被置于 "figure" 元素的第一个或最后一个子元素的位置。

    • <figure>标签规定独立的流内容（图像、图表、照片、代码等等）。

    • <footer>元素描述了文档的底部区域。
