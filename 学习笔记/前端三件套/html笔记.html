                    # 语义元素（semantic elements）

                    顾名思义，就是能表达出意义的元素，可以用来指明导航、头部、主体、侧边栏以及尾部。这些语义元素也都是块元素。

                    • <header>元素描述了文档的头部区域

                    • <nav>标签定义导航链接的部分。

                    • <section>标签定义文档中的节（section、区段）。比如章节、页眉、页脚或文档中的其他部分。

                    • <article>标签定义独立的内容。

                    • <aside>标签定义页面主区域内容之外的内容（比如侧边栏）。

                    • <figcaption>标签定义 <figure> 元素的标题，该元素应该被置于 "figure" 元素的第一个或最后一个子元素的位置。

                    • <figure>标签规定独立的流内容（图像、图表、照片、代码等等）。

                    • <footer>元素描述了文档的底部区域。

                    # 块元素（block）

                    1. 独占一行，自动换行
                    2. 可设置宽高和内外边距，默认是它容器（父级元素）的100%
                    3. 可以容纳其他块级元素和行内元素（内联元素）
                    4. 常见：div、p、h1、ol、ul、dl、tabel、form、语义元素

                    # 行内元素（inline）

                    1. 不独占一行，不自动换行
                    2. 设置宽、高无效，默认宽高是其自身内容的宽高。
                    3. 只能设置水平方向的内外边距。
                    4. 只能容纳其他行内元素和文本。
                    5. 行内元素的paddding可以设置,margin只能够设置水平方向的边距
                    6. 常见：a、span、i、strong、em、label、cite、code

                    # 行内块元素（inline-block）

                    1. 不独占一行，不自动换行
                    2. 其他特性和块元素一样
                    3. 常见：img、input、select、textarea、button
