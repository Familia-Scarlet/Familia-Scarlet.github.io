# WebAPI学习

# 基本认知

  1. DOM(文档对象模型)：包含各种HTML标签对应的JS对象，用于操作网页内容，可实现网页内容特效和用户交互。

  2. DOM树：将HTML文档以树形结构表现出来的一棵树，体现了标签与标签的关系。

  3. DOM对象：浏览器根据HTML文档生成的JS对象，有其相应的属性，与标签一一对应。

  4. document对象：DOM中提供的一个对象，是最大的对象，网页全部内容都在里面。该对象提供了操作网页内容的方法。

  5. BOM(浏览器对象模型)：整个window、navigator、location、document、history、screen对象组成浏览器对象模型，其中window是最大的BOM对象。

  6. window对象：浏览器内置的全局对象，BOM中最大的对象，所有的WebAPIs均基于window对象，window对象下包含了navigator、location、document、history、screen对象。

    • navigator：记录浏览器设备的信息。

    • location：控制浏览器页面位置。

    • document：记录浏览器内容结构。

    • history：记录浏览器浏览记录。

    • screen：记录用户的屏幕信息。
